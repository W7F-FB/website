# Project Rules

## Prismic / Slice Machine
- **Do NOT manually edit `prismicio-types.d.ts`** — types are auto-generated by Slice Machine. Any manual changes will be overwritten.
- **Update slice `model.json` files directly** — the Prismic MCP `save_slice_data` tool does not reliably write local model files. Edit `model.json` manually instead.
