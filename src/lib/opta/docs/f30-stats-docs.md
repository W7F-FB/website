# Season Statistics Appendices

## Team Stats

| Stat | Description |
|------|-------------|
| Games Played | Total number of games played |
| Aerial Duels | Aerial duels - the definition of an aerial duel is that it is between two players - either both jumping or only one jumping and the other contesting the header by making it difficult to win - and one player has to win the contest with a header OR if it is between an outfield player and a goalkeeper then the goalkeeper is allowed to punch it clear |

| Aerial Duels won | Duel where the ball is in the air and possession is won |

| Right Foot Goals | Goals scored via an attempt using the right foot |
| Set Pieces Goals | Goal from a direct free kick |

| Unsuccessful Crosses open play | Crosses made in open play which do not find a team-mate |

| Clearances Off the Line | Clearance off the goal line |

| defensive_actions | Number of defensive moves made by the team in the defensive third |
| PPDA | (Passes Per Defensive Action) measures how many opposition passes a team allows before making a defensive action, indicating pressing intensity—lower values mean stronger pressing. Note: Calculation formula = opposition_passes / defensive_actions |
